# Practicals / Experiments (Session 2025-26)

This folder contains a collection of practical programs covering HTML5, CSS3, JavaScript, and TypeScript.

## List of Practicals

1.  **[01-semantic-html-layout](./01-semantic-html-layout/index.html)**: HTML5 semantic elements and basic CSS layout.
2.  **[02-user-registration-form](./02-user-registration-form/index.html)**: User registration form with focus styles and box model.
3.  **[03-responsive-flex-grid](./03-responsive-flex-grid/index.html)**: Responsive web page using Flexbox (navigation) and Grid (content).
4.  **[04-js-basics](./04-js-basics/script.js)**: JS variables, data types, operators, and control flow.
5.  **[05-js-functions-arrays-objects](./05-js-functions-arrays-objects/script.js)**: Normal/arrow functions and array/object operations.
6.  **[06-js-event-handling](./06-js-event-handling/index.html)**: Dynamic DOM updates via click and input events.
7.  **[07-js-scope-closures](./07-js-scope-closures/script.js)**: JS execution context, scope, and closures.
8.  **[08-js-inheritance-es6](./08-js-inheritance-es6/script.js)**: Prototype-based inheritance and ES6 features.
9.  **[09-js-async-promises](./09-js-async-promises/script.js)**: Asynchronous execution using setTimeout, Promises, and async/await.
10. **[10-dom-manipulation](./10-dom-manipulation/index.html)**: DOM traversal and dynamic element manipulation.
11. **[11-event-delegation-debugging](./11-event-delegation-debugging/index.html)**: Event delegation and debugging practices.
12. **[12-ts-basics](./12-ts-basics/script.ts)**: TypeScript types, interfaces, and union types.
13. **[13-ts-generics-narrowing](./13-ts-generics-narrowing/script.ts)**: Generic functions and type narrowing in TypeScript.
14. **[14-ts-frontend-integration](./14-ts-frontend-integration/index.html)**: Integrating TypeScript into a frontend project with `tsconfig.json`.
